# Kindred HR Bot

An AI-powered Human Resources Assistant designed to automate routine HR operations, improve employee engagement, streamline communication, and provide intelligent workforce management solutions.

## Overview

Kindred HR Bot serves as a virtual HR professional, helping organizations manage recruitment, employee support, scheduling, onboarding, performance tracking, and administrative HR tasks through artificial intelligence.

The platform combines conversational AI, smart automation, and workforce analytics to reduce manual work and enhance productivity across the organization.

## Installation

### Clone Repository

```bash
git clone https://github.com/thapza48-ship-it/kindred-hr-bot.git
cd kindred-hr-bot
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file:

```env
DATABASE_URL=
OPENAI_API_KEY=
JWT_SECRET=
SMTP_HOST=
SMTP_PORT=
SMTP_USER=
SMTP_PASSWORD=
```

### Start Development Server

```bash
npm run dev
```
